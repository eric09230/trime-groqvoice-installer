洋蔥注音 配色工具

選 5 個顏色，下載自定義配色的 trime.yaml

  1. 下方選擇 5 個顏色，即時預覽鍵盤效果
  2. 按下載，自動產生配色好的 trime.yaml
  3. 放回手機 /sdcard/rime/ 覆蓋原檔
  4. Trime → 配置 → 立即同步使用者資料
  5. 進入配色，選擇「自定義配色」
